### Basic git command:
```bash
git clone url

git add hello.html

git commit -m "Add hello.html"
git commit -am "Add a heading"

git status
git pull
git push

git log
git reset --hard <commit>
git reset --hard origin/master

git branch
```
how to fix merge conflict:
> 1.Modify the file until you are satisfied;
> 2.git commit -m "Message"
> 3.git push
